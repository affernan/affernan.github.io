# Alejandro Fernandez Gil - Academic Website

This repository contains the source files for the personal academic website published through GitHub Pages at `https://affernan.github.io/`.

The site presents research interests, publications, teaching material, academic links, and supporting documents related to operations research, sustainable transportation, green vehicle routing, and energy systems.

## Repository Contents

- `index.html`: Main website page.
- `css/`: Stylesheet resources.
- `img/`: Images used by the website.
- `doc/`: Public documents linked from the site.
- `papers/`: Paper-related files made available through the website.
- `*.bib`: Bibliographic data used for publication lists.
- `robots.txt` and `sitemap.xml`: Basic indexing metadata.

## Development Notes

This is a static website. It can be previewed locally by opening `index.html` in a browser or by serving the repository with any static file server.

Changes pushed to the default branch are published by GitHub Pages according to the repository settings.
